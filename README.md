# ud923-course
Introduction to Operating Systems Udacity course examples

## Download
```
git clone https://github.com/smalinux/ud923-course
cd ud923-course
```

## Build and clean all lessons
```
make all
make clean
```
## How to use
```
cd Lesson01
./example_name      # all examples based on single-page style
```
